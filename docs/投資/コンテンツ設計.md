| **実績訴求**         | 「累計受講者 12,500 名」「満足度 94.3%」                       | 誇大表現・誤認表現・虚偽の数値             |
| **CTA 文言**         | 「まずは無料で体験してみる」「投資の第一歩を踏み出す」         | 「今すぐ申し込まないと損」「残りわずか！」 |
| **口コミ導入**       | 「受講後の変化をインタビュー」「実際の受講生の声」             | 「全員が成功」「必ず結果が出ます」         |
| **カリキュラム説明** | 「基礎から実践まで体系的に学べる」                             | 「これだけで完璧」「他は不要」             |
| **リスク表記**       | 「投資にはリスクが伴います。元本保証はありません」（必須）     | リスク表記の省略・曖昧な表現               |

### ライティング原則

1. **初心者目線を徹底**

   - 専門用語は必ず解説を添える
   - 「わからなくて当然」という前提で書く
   - 具体例・比喩を多用する

2. **ポジティブだが誠実に**

   - 「学べば判断できるようになる」（◯）
   - 「必ず儲かる」（×）
   - 希望を持たせつつ、現実的な期待値を示す

3. **行動を後押しする表現**

   - 「まずは」「第一歩」「無料で」など、ハードルを下げる言葉
   - 「あなたも」で共感を誘う
   - 「〜してみませんか？」と提案型にする

4. **法令遵守（金融商品取引法・景品表示法）**
   - 必ずリスク表記を入れる
   - 「絶対」「必ず」「100%」などの断定表現を避ける
   - 実績数値は裏付けを取る

## 4. 構成テンプレート（説得構造）

### 基本フロー：理解 → 共感 → 信頼 → 行動

```
1. Hero（ファーストインパクト）
   ↓ 「こういうサービスがあるんだ」（認知）

2. 実績・信頼（Authority）
   ↓ 「多くの人が利用しているんだ」（社会的証明）

3. 学べること（Value Proposition）
   ↓ 「自分の課題が解決できそう」（価値理解）

4. カリキュラム／教材（Specificity）
   ↓ 「具体的にどう学ぶかわかった」（期待形成）

5. 受講の流れ（Process）
   ↓ 「手順がわかって安心」（不安軽減）

6. 口コミ／メディア掲載（Social Proof）
   ↓ 「自分と似た人が成功している」（共感・自己投影）

7. よくある質問（Objection Handling）
   ↓ 「疑問点が解消された」（障壁除去）

8. 最終 CTA（Action）
   ↓ 「じゃあ申し込んでみよう」（行動）
```

### ページタイプ別構成テンプレート

#### LP 型（トップページ・セミナー LP）

```
1. Hero（キャッチ + CTA）
2. 課題提起（ターゲットの悩み）
3. 解決策提示（このスクールで解決できる）
4. 実績・信頼
5. 具体的な内容（カリキュラム・教材）
6. ベネフィット（得られる未来）
7. 口コミ・成功事例
8. FAQ
9. 最終 CTA
```

#### 詳細型（カリキュラム・教材紹介）

```
1. ページヒーロー
2. 概要説明（何が学べるか）
3. 詳細内容（具体的な構成）
4. 特徴・強み
5. サポート・付加価値
6. 料金・規約（該当ページのみ）
7. CTA
```

#### 一覧型（スケジュール・受講生の声）

```
1. ページタイトル
2. 絞り込み検索（該当ページ）
3. コンテンツ一覧（カード or テーブル）
4. ページネーション
5. 下部 CTA
```
