## 1. デザインコンセプト

### コアコンセプト：**信頼 × 学習**

投資という金融商品を扱うスクールとして、信頼感・誠実さ・専門性を第一に。
初心者が「ここなら安心して学べる」と感じられるデザインを目指します。

### デザインの方向性

| 要素         | 詳細                                                   |
| ------------ | ------------------------------------------------------ |
| **テーマ**   | 信頼できる教育機関                                     |
| **印象**     | 清潔感・誠実・知的・落ち着き                           |
| **トーン**   | 白ベース／ネイビーブルーアクセント／ミニマル           |
| **目的**     | 安心して申込できる雰囲気、初心者でも親しみやすい UI    |
| **回避要素** | 派手な色使い・過度なアニメーション・ギャンブル的な演出 |

### 競合との差別化

- **シンプルで洗練されたデザイン**：情報過多にせず、必要な情報を段階的に提示
- **白とネイビーの 2 色構成**：視覚的なノイズを減らし、CTA を際立たせる
- **余白を活かしたレイアウト**：読みやすさと上品さを両立
- **初心者目線の UI**：複雑な操作を排除、直感的なナビゲーション

## 2. カラーパレット

### 基本カラー

| 用途                             | カラー名       | HEX コード | RGB           | Tailwind CSS            | 使用場所                   |
| -------------------------------- | -------------- | ---------- | ------------- | ----------------------- | -------------------------- |
| **Primary（アクセント）**        | ネイビーブルー | `#1A3C78`  | 26, 60, 120   | `bg-accent text-accent` | CTA ボタン・見出し・リンク |
| **Background（基本背景）**       | 白             | `#FFFFFF`  | 255, 255, 255 | `bg-white`              | 全体ベース                 |
| **Section BG（セクション背景）** | 薄グレー       | `#F9FAFB`  | 249, 250, 251 | `bg-gray-50`            | 交互背景・カード背景       |
| **Text（メインテキスト）**       | ダークグレー   | `#1F2937`  | 31, 41, 55    | `text-gray-800`         | 本文・見出し               |
| **Sub Text（補足テキスト）**     | グレー         | `#6B7280`  | 107, 114, 128 | `text-gray-600`         | 補足説明・キャプション     |
| **Border（境界線）**             | 薄グレー       | `#E5E7EB`  | 229, 231, 235 | `border-gray-200`       | セクション区切り・カード枠 |

### アクセントカラー（状態表示用）

| 用途        | カラー名 | HEX コード | Tailwind CSS     | 使用場所                      |
| ----------- | -------- | ---------- | ---------------- | ----------------------------- |
| **Success** | グリーン | `#22C55E`  | `text-green-500` | 実績バッジ・残席 ◯ 表示       |
| **Warning** | オレンジ | `#F59E0B`  | `text-amber-500` | 残席 △ 表示・注意事項         |
| **Error**   | レッド   | `#EF4444`  | `text-red-500`   | 満席 × 表示・エラーメッセージ |

### カラー使用ルール

1. **背景は白とグレー 50 の 2 色のみ**：交互配置で視認性向上
2. **CTA ボタンは必ずネイビーブルー**：他の要素では使用しない
3. **テキストは 2 階層**：メイン（gray-800）とサブ（gray-600）
4. **状態表示のみアクセントカラー使用**：緑・黄・赤は控えめに

## 3. タイポグラフィ

### フォントファミリー

```css
/* Google Fonts 読み込み */
@import url("https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;600;700&display=swap");

/* Tailwind CSS 設定 */
fontfamily: {
  sans: [ "Noto Sans JP", "sans-serif"];
}
```

### フォントサイズ・ウェイト一覧

| 要素                     | PC サイズ | SP サイズ | ウェイト | Tailwind CSS クラス                  | 使用場所           |
| ------------------------ | --------- | --------- | -------- | ------------------------------------ | ------------------ |
| **H1（ページタイトル）** | 40px      | 28px      | Bold     | `text-4xl md:text-5xl font-bold`     | トップヒーロー     |
| **H2（大見出し）**       | 32px      | 24px      | Semibold | `text-3xl md:text-4xl font-semibold` | セクション見出し   |
| **H3（中見出し）**       | 24px      | 20px      | Semibold | `text-2xl md:text-3xl font-semibold` | サブセクション     |
| **H4（小見出し）**       | 20px      | 18px      | Medium   | `text-xl md:text-2xl font-medium`    | カード内見出し     |
| **Body（本文）**         | 16px      | 16px      | Regular  | `text-base leading-relaxed`          | 本文・説明文       |
| **Small（補足）**        | 14px      | 14px      | Regular  | `text-sm text-gray-600`              | キャプション・注釈 |
| **Button（ボタン）**     | 16px      | 16px      | Semibold | `text-base font-semibold`            | CTA ボタン         |

### 行間・字間

```css
/* 本文 */
line-height: 1.75; /* leading-relaxed */
letter-spacing: 0.025em; /* tracking-wide */

/* 見出し */
line-height: 1.25; /* leading-tight */
letter-spacing: -0.025em; /* tracking-tight */
```

### タイポグラフィルール

1. **見出しは階層を守る**：H1 → H2 → H3 の順で使用（飛ばさない）
2. **本文は 16px 固定**：可読性を最優先
3. **行間はゆったり**：1.75（leading-relaxed）で読みやすく
4. **長文は最大幅制限**：`max-w-3xl`（48rem = 768px）で 1 行が長すぎないように

## 4. レイアウトルール

### レスポンシブブレークポイント

```js
// tailwind.config.js
screens: {
  'sm': '640px',   // スマートフォン（横向き）
  'md': '768px',   // タブレット
  'lg': '1024px',  // PC
  'xl': '1280px',  // 大画面 PC
}
```

### コンテナ設定

| デバイス       | 最大幅           | Tailwind クラス           | 備考             |
| -------------- | ---------------- | ------------------------- | ---------------- |
| **SP**         | 100%（余白付き） | `px-4`                    | 左右 16px の余白 |
| **タブレット** | 100%（余白付き） | `px-6 md:px-8`            | 左右 32px の余白 |
| **PC**         | 1024px           | `max-w-screen-lg mx-auto` | 中央寄せ         |
| **大画面**     | 1280px           | `max-w-screen-xl mx-auto` | 超ワイド画面対応 |

### セクション間の余白

```html
<!-- SP：上下 48px、PC：上下 64px -->
<section class="py-12 md:py-16">
  <!-- SP：上下 64px、PC：上下 96px（重要セクション） -->
  <section class="py-16 md:py-24"></section>
</section>
```

### グリッドレイアウト

```html
<!-- 1カラム（SP） → 2カラム（タブレット） → 3カラム（PC） -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <!-- 1カラム（SP） → 2カラム（PC） -->
  <div class="grid grid-cols-1 lg:grid-cols-2 gap-8"></div>
</div>
```

### SP ファースト設計の原則

1. **モバイルから設計**：デフォルトは SP、`md:` `lg:` で拡張
2. **タッチ領域は 44px 以上**：ボタン・リンクは指で押しやすく
3. **横スクロール禁止**：`overflow-x-hidden` で横スクロール防止
4. **フォームは縦積み**：入力項目は SP で 1 カラム
5. **画像は可変**：`w-full h-auto` で親要素に追従

#### カウントアップアニメーション（オプション）

```tsx
// useCountUp.ts（カスタムフック）
import { useEffect, useState } from "react"

export const useCountUp = (end: number, duration: number = 2000) => {
  const [count, setCount] = useState(0)

  useEffect(() => {
    let startTime: number
    const animate = (currentTime: number) => {
      if (!startTime) startTime = currentTime
      const progress = (currentTime - startTime) / duration

      if (progress < 1) {
        setCount(Math.floor(end * progress))
        requestAnimationFrame(animate)
      } else {
        setCount(end)
      }
    }

    requestAnimationFrame(animate)
  }, [end, duration])

  return count
}
```

### 5-5. FAQ アコーディオン

```tsx
// FAQAccordion.tsx（React コンポーネント）
"use client"

import { useState } from "react"
import { ChevronDown } from "lucide-react"

interface FAQItem {
  question: string
  answer: string
}

export const FAQAccordion = ({ items }: { items: FAQItem[] }) => {
  const [openIndex, setOpenIndex] = useState<number | null>(null)

  return (
    <div class="space-y-4">
      {items.map((item, index) => (
        <div key={index} class="border-b border-gray-200">
          {/* 質問（クリック可能） */}
          <button
            onClick={() => setOpenIndex(openIndex === index ? null : index)}
            class="flex justify-between items-center w-full py-4 text-left 
                   hover:text-accent transition-colors duration-200"
          >
            <span class="text-lg font-semibold text-gray-800">
              {item.question}
            </span>
            <ChevronDown
              class={`w-5 h-5 text-accent transition-transform duration-200 
                      ${openIndex === index ? "rotate-180" : ""}`}
            />
          </button>

          {/* 回答（展開式） */}
          {openIndex === index && (
            <div class="pb-4 text-gray-600 leading-relaxed animate-fadeIn">
              {item.answer}
            </div>
          )}
        </div>
      ))}
    </div>
  )
}
```

// useCTABar.ts
"use client"

import { useEffect, useState } from "react"

export const useCTABar = (threshold: number = 200) => {
  const [isVisible, setIsVisible] = useState(false)

  useEffect(() => {
    const handleScroll = () => {
      setIsVisible(window.scrollY > threshold)
    }

    window.addEventListener("scroll", handleScroll)
    return () => window.removeEventListener("scroll", handleScroll)
  }, [threshold])

  return isVisible
}

// useScrollAnimation.ts
"use client"

import { useEffect, useRef, useState } from "react"

export const useScrollAnimation = () => {
  const ref = useRef<HTMLDivElement>(null)
  const [isVisible, setIsVisible] = useState(false)

  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          setIsVisible(true)
          observer.disconnect()
        }
      },
      { threshold: 0.1 }
    )

    if (ref.current) {
      observer.observe(ref.current)
    }

    return () => observer.disconnect()
  }, [])

  return { ref, isVisible }
}

// 使用例
const Component = () => {
  const { ref, isVisible } = useScrollAnimation()

  return (
    <div
      ref={ref}
      className={`transition-all duration-700 ${
        isVisible ? "opacity-100 translate-y-0" : "opacity-0 translate-y-10"
      }`}
    >
      コンテンツ
    </div>
  )
}
