## 📋 プロジェクト概要

このプロジェクトは、株式投資スクールの無料体験セミナー申込を月間 100 件獲得することを目的とした Web サイトの開発です。

### 技術スタック

- **フレームワーク**: Next.js 14 (Pages Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS v4
- **デプロイ**: Vercel
- **データ管理**: ローカル JSON ファイル（CMS 不使用）
- **状態管理**: React Hooks
- **フォーム**: React Hook Form + Zod
- **アイコン**: Lucide React

## 🎯 開発の基本方針

### 1. 設計ドキュメントの完全遵守

以下のドキュメントに記載された全ての仕様を厳密に守ること：

- **要件定義.md**: ビジネス要件、ターゲット、KPI
- **情報設計\_サイトマップ.md**: サイト構造、ページ構成、導線設計
- **コンテンツ設計.md**: 各ページの詳細コンテンツ、文言
- **デザイン指針.md**: デザインシステム、カラー、タイポグラフィ、コンポーネント設計
- **データ設計.md**: データモデル、TypeScript 型定義、JSON スキーマ
- **ルーティング設計.md**: URL 構造、ページ骨子、コンポーネント構成

### 2. 品質基準

#### パフォーマンス

- **LCP（Largest Contentful Paint）**: 2.5 秒以内
- **FID（First Input Delay）**: 100ms 以内
- **CLS（Cumulative Layout Shift）**: 0.1 以下
- **モバイルページ速度スコア**: 85 以上（PageSpeed Insights）

#### アクセシビリティ

- **WCAG 2.1 AA 準拠**を目標
- キーボード操作対応必須
- スクリーンリーダー対応（aria-label、alt 属性の適切な設定）
- 色のコントラスト比 4.5:1 以上
- タッチ領域 44px 以上

#### SEO

- title/meta description 最適化（各ページ固有）
- 構造化データ対応（Course、Organization、FAQPage）
- OGP 設定（SNS シェア対応）
- セマンティック HTML（適切な見出し階層）

#### レスポンシブデザイン

- **SP ファースト設計**: モバイルから設計し、デスクトップに拡張
- ブレークポイント: sm(640px), md(768px), lg(1024px), xl(1280px)
- 横スクロール禁止
- 画像は可変（w-full h-auto）

### 3. FAQAccordion（よくある質問）

```tsx
// components/sections/FAQAccordion.tsx
"use client"

import { useState } from "react"
import { ChevronDown } from "lucide-react"

interface FAQItem {
  question: string
  answer: string
}

export function FAQAccordion({ items }: { items: FAQItem[] }) {
  const [openIndex, setOpenIndex] = useState<number | null>(null)

  return (
    <div className="space-y-4">
      {items.map((item, index) => (
        <div key={index} className="border-b border-gray-200">
          <button
            onClick={() => setOpenIndex(openIndex === index ? null : index)}
            className="flex justify-between items-center w-full py-4 text-left 
                     hover:text-accent transition-colors duration-200"
          >
            <span className="text-lg font-semibold text-gray-800">
              {item.question}
            </span>
            <ChevronDown
              className={`w-5 h-5 text-accent transition-transform duration-200 
                        ${openIndex === index ? "rotate-180" : ""}`}
            />
          </button>
          {openIndex === index && (
            <div className="pb-4 text-gray-600 leading-relaxed">
              {item.answer}
            </div>
          )}
        </div>
      ))}
    </div>
  )
}
```

// components/layouts/CTAStickyBar.tsx
"use client"

import { useEffect, useState } from "react"
import { Phone } from "lucide-react"

export function CTAStickyBar() {
  const [isVisible, setIsVisible] = useState(false)

  useEffect(() => {
    const handleScroll = () => {
      setIsVisible(window.scrollY > 200)
    }

    window.addEventListener("scroll", handleScroll)
    return () => window.removeEventListener("scroll", handleScroll)
  }, [])

  if (!isVisible) return null

  return (
    <div
      className="fixed bottom-0 left-0 right-0 z-50 md:hidden 
                  bg-accent text-white shadow-lg"
    >
      <div className="flex items-center justify-between px-4 py-3">
        <a
          href="tel:0120-xxx-xxx"
          className="flex items-center gap-2 text-sm font-semibold"
        >

### データ読み込みユーティリティ

```typescript
// lib/data-loader.ts
import fs from "fs"
import path from "path"

export function loadData<T>(filename: string): T[] {
  const filePath = path.join(process.cwd(), "src/data", filename)
  const fileContents = fs.readFileSync(filePath, "utf8")
  return JSON.parse(fileContents)
}

// 使用例（getStaticPropsで使用）
export function getEvents(): Event[] {
  return loadData<Event>("events.json")
}

export function getFeaturedReviews(limit: number = 3): Review[] {
  const reviews = loadData<Review>("reviews.json")
  return reviews.filter((r) => r.featured).slice(0, limit)
}
```
