- サロン名：{例：Y'm 整体サロン}

- 立地・アクセス：{例：戸田市、最寄り駅、駐車場有無}

## 🧩 出力フォーマット

### 1. 目的（Purpose）

- このサイトを通じて何を実現するか（数値 KPI があれば記載）
- 競合サロンとの差別化ポイント
- プレオープン戦略（該当する場合）

### 2. ターゲット（Persona）

- 年齢層・属性・居住地域・職業
- 抱えている体の悩み・課題
- 来訪前の心理状態／求めている価値
- サロン利用経験・予算感

### 3. コンバージョン目標（Goal）

- 一次 CV（LINE 予約・電話予約など）
- 二次 CV（LINE 友だち登録／Instagram フォロー／資料請求など）
- 予約率・来店率・リピート率の KPI（仮値）

### 4. 機能要件（Functional Requirements）

| 機能カテゴリ         | 内容                                           | 備考                  |
| -------------------- | ---------------------------------------------- | --------------------- |
| LP 構成              | ヒーロー、サロン特徴、メニュー、料金、予約 CTA | トップページ          |
| メニュー・料金ページ | 整体／エステ／電磁パルス詳細                   | PDF メニューの Web 化 |
| AI 姿勢診断紹介      | 診断プロセス、ビフォーアフター事例             | 科学的根拠の訴求      |
| 予約システム         | LINE 予約連携、電話予約、日時・コース選択      | LINE Bot 連携想定     |
| お客様の声           | 施術前後写真、年齢・職業・改善内容             | 許可取得済み前提      |
| スタッフ・サロン紹介 | 施術者プロフィール、サロン内観、設備           | 安心感・信頼性向上    |
| アクセス・店舗情報   | Google マップ、道順、駐車場、営業時間          | 来店ハードル軽減      |
| FAQ                  | 料金・施術内容・予約方法・AI 診断について      | 不安解消              |

### 5. 非機能要件（Non-functional）

- 速度：LCP < 2.5s（特にモバイル重視）
- SEO：ローカル SEO 対策、Google マイビジネス連携
- 保守：CMS で更新可能（メニュー・料金・お客様の声）
- セキュリティ：reCAPTCHA、個人情報保護、写真使用許可管理
- アクセシビリティ：WCAG 2.1 AA 準拠を目標

### 6. 成功指標（KPI）

| 指標               | 目標値    | 測定方法         |
| ------------------ | --------- | ---------------- |
| 予約申込率         | 4〜6%     | GA4              |
| LP 滞在時間        | 90 秒以上 | GA4              |
| 予約完了率         | 70%以上   | GA4 ファネル     |
| 来店率             | 85%以上   | 予約管理システム |
| リピート率         | 60%以上   | 顧客管理システム |
| モバイル速度スコア | 85 以上   | PageSpeed        |

### 7. リスク・留意点

- 医療法・薬機法：治療効果表現の規制（「治る」「治療」禁止）
- 景品表示法：ビフォーアフター写真使用時の個人差明記
- 個人情報保護：写真使用許可、プライバシーポリシー整備
- 特定商取引法：料金表示、キャンセル規定明記
- 競合との差別化ポイント維持
- SNS 炎上対策（適切な投稿ガイドライン）

### 8. 今後の確認アクション

- ターゲット顧客の検索キーワード調査（「AI 姿勢診断 整体」等）
- 競合サロンベンチマーク（Re:Bone、カラダファクトリー、地域サロン 3 社）
- 予約導線 AB テスト案出し（LINE vs 電話 vs フォーム）
- お客様の声・写真素材収集（許可取得）
- LINE 公式アカウント・予約システム選定

### 9. デザイン・ブランディング方針

- カラーパレット：{例：ホワイト＋ベージュ＋薄グリーン}
- フォント：{例：Noto Sans JP（安心感）＋ Roboto（AI 技術感）}
- 写真表現：「データ × リラックス」を意識、自然光撮影
- CTA 配置：各セクション末＋フローティングボタン

### 10. マーケティング戦略

- プレオープン特別価格の訴求方法
- ローカル SEO 対策（地域名＋サービス名）
- SNS 運用方針（Instagram 中心、施術風景・ビフォーアフター）
- 口コミ・紹介制度の設計
